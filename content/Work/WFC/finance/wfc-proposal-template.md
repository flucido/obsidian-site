# WFC Proposal Template

> Generate proposals from this template after dossier approval and consult.
> All proposals are DRAFTS until human review and approval.
> Prices reference [[Work/Shared/pricing-catalog\\|Pricing Catalog]].

---

## Proposal Workflow

1. **Trigger:** Consult completed, offer fit confirmed
2. **Draft:** Populate proposal from this template
3. **Review:** Human reviews and approves pricing, scope, terms
4. **Send:** Human triggers send to prospect
5. **Track:** Move lead to "Proposed" stage in [[WFC Pipeline]]

---

## Proposal Document Structure

### Header
```
WellFull Collective
[Client Name] — [Practice Name]
Proposal Date: [YYYY-MM-DD]
Valid Through: [YYYY-MM-DD + 30 days]
```

### 1. Understanding
Brief restatement of what was discussed in consult:
- Practice type, specialties, target clients
- Current website situation (platform, pain points)
- Goals for new website

### 2. Recommended Solution
| Element | Details |
|---------|---------|
| Offer Tier | WFC-0[1/2/3] — [Landing Page / CMS Site / Web Application] |
| Pages | [list specific pages] |
| Key Features | [booking, intake, blog, resource library, etc.] |
| Design Approach | [nervous-system-aware, specific brand direction] |
| Therapeutic Brand Positioning | Included |

### 3. Investment
| Item | Price |
|------|-------|
| [Offer tier name] | $[from Pricing Catalog] |
| [Add-on 1 if applicable] | $[from Pricing Catalog] |
| [Add-on 2 if applicable] | $[from Pricing Catalog] |
| **Total One-Time** | **$[sum]** |
| Monthly Support (optional) | $[100–250]/month |

### 4. Timeline
| Phase | Duration | Deliverable |
|-------|----------|-------------|
| Discovery + Brand Positioning | 1 week | Brand direction, content strategy |
| Design | 1–2 weeks | Design mockups, review cycle |
| Build | 1–2 weeks | Development, content loading |
| Review + Launch | 1 week | Final review, go-live |
| **Total** | **4–6 weeks** | |

### 5. Payment Terms
- 50% deposit to begin work ($[amount])
- 50% on launch ($[amount])
- Monthly support (if selected): $[amount]/month, first payment on launch

### 6. What's Included
- Therapeutic Brand Positioning session
- Design system tokens (colors, typography, spacing)
- Mobile-responsive, WCAG AA accessible
- Basic SEO (meta, structured data, sitemap)
- 30 days post-launch support

### 7. Next Steps
1. Review and sign proposal
2. 50% deposit invoice will be sent
3. Discovery call scheduled within 1 week of deposit

## Anti-Patterns
- Do NOT quote prices before consult
- Do NOT send proposal without human approval
- Do NOT customize scope without updating this template reference
