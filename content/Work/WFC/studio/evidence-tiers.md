# Evidence Tier Classification Guide

## Purpose
Every piece of research entering the NeroDesign system must be classified into one of three evidence tiers. This prevents treating single observations as laws, and keeps our methodology honest about the origin of each design rule.

## Tier 1 — Strong Evidence
**Definition:** Replicated findings, quantitative results, peer-reviewed consensus.

**Criteria:**
- Published in peer-reviewed journals with replication
- Quantitative results with statistical significance
- Multiple independent studies converging on the same finding
- Established theoretical frameworks with broad acceptance

**Examples from corpus:**
- P2 component timing (200-400ms) for early aesthetic classification
- LPP component timing (400-600ms) for deep emotional arousal
- PAD model three-dimensional structure (Pleasure, Arousal, Dominance)
- SVM emotion classification success rates (72%+ in quasi-experimental settings)
- EEG frequency band definitions (Delta, Theta, Alpha, Beta, Gamma)

**Design rule weight:** HIGH — can directly justify studio design decisions.

## Tier 2 — Interpretive Frameworks
**Definition:** Models that are useful but context-dependent, requiring adaptation.

**Criteria:**
- Proposed in peer-reviewed work but not yet widely replicated
- Context-specific findings that may not generalize
- Frameworks requiring interpretation for new domains
- Single-study observations with plausible mechanisms

**Examples from corpus:**
- 17-dimensional AHD design matrix
- Neuro-static user archetypes (age/income-based emotional traits)
- Sublimation theory for negative emotional engagement
- 9-17 element layout constraint
- Museum visitor emotional engagement protocol

**Design rule weight:** MEDIUM — useful as guidance, requires studio validation before becoming a rule.

## Tier 3 — Studio Methodology
**Definition:** Original synthesis derived by the studio from Tier 1 and Tier 2 sources.

**Criteria:**
- Derived from higher-tier evidence through studio interpretation
- Represents the studio's unique methodological contribution
- Must cite its Tier 1 or Tier 2 source(s)
- Subject to revision as new evidence emerges

**Examples (to be developed):**
- "Revisability reduces impulse-anxiety in high-sensitivity contexts"
- "Design should regulate before it converts"
- "Every form, CTA, and navigation action must support undo, draft, or return paths"

**Design rule weight:** VARIABLE — depends on the strength of the source evidence chain.

## Classification Process
1. Read the source material
2. Identify the specific claim or finding
3. Ask: Is this replicated? Quantitative? Peer-reviewed consensus?
4. If yes to all → Tier 1
5. If useful but context-dependent → Tier 2
6. If our own synthesis → Tier 3 (with source citation)
7. Record in the Evidence Ledger with tier, source, and design implication
