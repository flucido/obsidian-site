# NeroDesign Client Deliverables

## What Clients Receive

### 1. Live Website
A fully functional, responsive website built to the NeroDesign system standards.
- All pages pass the six-gate Critique Heuristics framework
- WCAG AA accessibility compliance minimum
- prefers-reduced-motion support
- Mobile-first responsive design
- Performance budget: LCP under 2.5s, CLS under 0.1

### 2. Design System Token File
A documented set of design tokens specific to the client's brand, derived from the NeroDesign system.
- Color palette (calm, engagement, and trust variants)
- Spacing scale
- Typography scale
- Motion rules
- All tokens calibrated to the client's target PAD coordinates
- Delivered as YAML + CSS custom properties

### 3. Emotional Journey Map
A page-by-page map of target emotional states and how the design supports them.
- Each page's target PAD coordinates
- Design rationale linking tokens to emotional targets
- Evidence references for key decisions
- User journey emotional arc (where arousal increases, where it decreases)

### 4. Critique Report
A completed Critique Heuristics evaluation for every page in the delivered website.
- All six gates evaluated
- Pass/Fail/N/A for each check
- Notes on any N/A decisions
- Signed off by the studio founder

### 5. Optional: Ongoing Measurement and Iteration
For clients who want continuous improvement:
- Quarterly critique re-evaluation
- User testing with high-sensitivity participants
- PAD coordinate validation through user feedback
- Token adjustments based on real-world usage
- Annual evidence ledger update with new research
