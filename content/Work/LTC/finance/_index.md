```
                                                │
                                                ▼

```

## District-Specific Notes
- School districts typically pay Net-30 via purchase order (PO)
- Large districts may require vendor registration before proposal
- California Ed Code procurement rules apply for districts over certain thresholds
- FERPA / AB 1584 compliance language required in all contracts
